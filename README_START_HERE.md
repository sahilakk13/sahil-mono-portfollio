# Mōno-style Portfolio Codex Context Pack

Use this folder as the context pack for Codex. Upload or paste these files before coding.

## Start order

1. Paste `codex-prompts/01_MASTER_CODEX_PROMPT.md` into Codex.
2. Upload or paste the full `context/` folder.
3. Upload or paste the full `data/` folder.
4. Ask Codex to create the React + Vite project and implement the site.
5. After the first build, paste `qa/01_STRICT_VISUAL_QA_FIX_PROMPT.md`.
6. If it still looks generic, paste `qa/02_NOT_CLOSE_ENOUGH_FIX_PROMPT.md`.

## Main reference

Template reference: https://monof-template.webflow.io/

Important: recreate the layout, behavior, spacing, motion feel, and responsive design. Do not copy proprietary Webflow source code or paid template assets.

## Expected output

A complete React + Vite + TypeScript + Tailwind + Framer Motion portfolio with:

- Home page matching the Mōno-style structure
- Studio page
- Three work layout pages
- Project detail pages
- Three blog layout pages
- Blog post pages
- Three contact pages
- Responsive header/mobile nav
- Footer/newsletter
- Smooth motion
- Data-driven content
- Vercel-ready build

## Do not allow Codex to do this

- Do not create a normal generic portfolio.
- Do not use colorful gradients.
- Do not skip inner pages.
- Do not remove animations.
- Do not add GitHub links unless later requested.
- Do not mention internships or unnecessary company names.
- Do not use random stock-template section layouts.
