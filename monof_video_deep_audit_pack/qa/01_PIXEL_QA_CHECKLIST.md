# Pixel QA Checklist After Codex Rebuild

Use this checklist with the reference video contact sheet.

## Navigation
- [ ] Top nav is minimal/absolute, not cloudy sticky.
- [ ] Bottom nav is hidden in hero.
- [ ] Bottom nav appears after hero/work start.
- [ ] Bottom nav does not cover footer content.

## Hero
- [ ] Central white hero card exists.
- [ ] Image/collage backdrop visible around card.
- [ ] Hairline grid and dots match reference feeling.
- [ ] Split bottom title is huge and cropped intentionally.
- [ ] Curved transition into partners exists.

## Work
- [ ] Left rail sticky.
- [ ] Right cards are a single vertical stack.
- [ ] Work cards are image-led and editorial, not placeholder geometry.
- [ ] Hover/cursor states work.

## Vision
- [ ] Long black pinned section.
- [ ] Text changes over scroll, not static.
- [ ] Top label and bottom dot/scroll label remain visible.
- [ ] Final media reveal exists.

## Services
- [ ] Huge row list.
- [ ] All rows visible.
- [ ] Active/inactive state uses pale/black contrast.
- [ ] Images/descriptions integrated with rows.

## Showreel
- [ ] Long black pinned section.
- [ ] Central media tile with surrounding tiles.
- [ ] Heading centered over media.

## Pricing/FAQ
- [ ] Pricing cards side-by-side.
- [ ] FAQ is 2×2 grid.
- [ ] Accordions open smoothly.

## Testimonials
- [ ] Long pinned light-gray section.
- [ ] Card changes by scroll progress.
- [ ] Counter updates 01/04 etc.
- [ ] Background floating cards exist.

## Stats/Success
- [ ] Black section.
- [ ] Metric/quote card rhythm.
- [ ] Success story has left image + right quote.

## Footer
- [ ] Contact image background + form overlay.
- [ ] Dense footer columns.
- [ ] Floating nav safe behavior.

## Build
- [ ] `npm run build` passes.
- [ ] No horizontal scroll at 1440.
- [ ] No card overlap.
