## Download Note

The Codex sandbox could not resolve external hosts while running `curl`, so the files may need to be downloaded locally with:

```bash
npm run download:media
```

The production components currently use the remote free-media URLs as fallbacks. The `local` paths below are the intended saved filenames in `public/media/`.
